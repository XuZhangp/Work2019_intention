# HEM
This repo contains the implementation of "Deep hierarchical encoding model for sentence semantic matching" in Keras & Tensorflow.
# Usage for python code
## 0. Requirement
python 3.6  
numpy==1.16.4  
pandas==0.22.0  
tensorboard==1.12.0  
tensorflow-gpu==1.12.0  
keras==2.2.4
